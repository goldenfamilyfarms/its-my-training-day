# Practice Exercises - React/Node.js Full Stack

## Overview

Hands-on exercises to validate your understanding of React and Node.js concepts. Each exercise provides a problem statement without the solution visible, allowing you to practice recreating concepts independently.

## How to Use

1. Read the problem file thoroughly
2. Attempt the solution without looking at hints
3. If stuck, use the progressive hints
4. Compare your solution with the reference solution
5. Note any gaps in your understanding for review

## Exercise Format

Each exercise consists of:
- **Problem file** (`{exercise}-problem.md`): Requirements and optional hints
- **Solution file** (`{exercise}-solution.md`): Reference implementation with explanation
