# Senior Full Stack Engineer - Sample JD

## Company: TechCorp (Sample)

## Role Overview
We're looking for a Senior Full Stack Engineer to build and maintain our customer-facing web applications and internal tooling.

## Requirements

### Must Have
- 5+ years of experience with React and Node.js
- Strong understanding of TypeScript
- Experience with state management (Redux, Zustand, or similar)
- Proficiency in RESTful API design
- Experience with PostgreSQL or similar relational databases
- Understanding of CI/CD pipelines and automated testing

### Nice to Have
- Experience with GraphQL
- Knowledge of containerization (Docker, Kubernetes)
- Familiarity with AWS services (Lambda, S3, RDS)
- Experience with real-time applications (WebSockets)

## Responsibilities
- Design and implement new features for our React-based frontend
- Build and maintain Node.js backend services
- Write comprehensive unit and integration tests
- Participate in code reviews and architectural discussions
- Mentor junior developers

## Interview Process
1. Technical phone screen (React/Node.js fundamentals)
2. Coding exercise (live coding)
3. System design discussion
4. Behavioral interview
